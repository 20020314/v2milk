# V2Milk
V2Ray 跨平台定制客户端

# 编写中
* Mac部分完成
* Win开发中
* Linux开发中
* 自定义节点开发中
* 二维码开发中

# 特点
* 使用Electron编写
* 完全开源，真实的开源，你不止能下载到README.md
* 可对接V2raySocks插件

# 使用方式
* 请修改 APP/src/main/config.js 里面的对应项目！
